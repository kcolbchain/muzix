# Contributing to Muzix

Muzix is an OP Stack L1 for music finance. We welcome contributors from both blockchain and music industries.

## Quick Start

1. Browse [open issues](https://github.com/kcolbchain/muzix/issues) tagged `good-first-issue` or `help-wanted`
2. Comment to claim, fork, branch, PR
3. Get reviewed within 48 hours

## What We Need

- **Solidity:** MUSD stablecoin contracts, royalty split logic, catalog tokenization
- **Go:** OP Stack node customization, sequencer config
- **TypeScript:** SDK for music apps, frontend components
- **Research:** Music finance economics, tokenization models, oracle design
- **Music industry:** Domain expertise, artist onboarding flows, rights management

## Contributor Tiers

| PRs Merged | Tier | Access |
|------------|------|--------|
| 0 | Explorer | Public issues |
| 1+ | Contributor | Paid bounties |
| 3+ | Core | Client projects, contract work |
