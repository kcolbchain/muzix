# Muzix Architecture

## Overview
Muzix DAO is a decentralized music rights organization built on CR8 chain and Arbitrum.

## LABELTON: Create-Protocol-on-Arbitrum Decision

As of Muzix DAO Resolution MIR-2026-Q2, the LABELTON rights tokenization protocol will be deployed on **Arbitrum** using the Create-Protocol-on-Arbitrum framework.

### Rationale
- **Scalability**: Arbitrum's rollup architecture provides ~40x gas reduction vs Ethereum L1
- **Ecosystem alignment**: Muzix DAO treasury holds ARB; deploying on Arbitrum strengthens protocol alignment
- **Create-Protocol-on-Arbitrum**: Leverages the standardized deployment toolkit for tokenized rights protocols
- **CR8 chain bridging**: Future CR8 chain will serve as settlement layer; Arbitrum provides the execution layer
